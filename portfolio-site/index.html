About

A little background

I'm an AWS Cloud and Security Engineer based in Alberta, Canada. I design, build, and harden cloud infrastructure on AWS — with a particular interest in identity, access, and compliance automation.

My focus right now is shipping a series of hands-on AWS projects, each one centered on a real-world security or architecture problem. The site you're reading is project one in that series.

I hold the AWS Solutions Architect – Associate certification and am working toward the AWS Security Specialty. When I'm not in the console, I'm reading the AWS Well-Architected Framework, the CIS Benchmarks, or whatever incident postmortem just dropped.
